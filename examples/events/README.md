# Events

Typed event listeners. Handler arguments are inferred from discord.js
`ClientEvents`.

- [`basic.ts`](./basic.ts) — positional and object forms; `once`.
- [`error-handling.ts`](./error-handling.ts) — thrown/rejected handlers route to the client `error` event.

See also: [file-based-loading](../file-based-loading), [getting-started](../getting-started).
