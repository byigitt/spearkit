# Select menus

All five select types. Each context exposes the relevant resolved data plus the
raw `ctx.values`.

- [`string-select.ts`](./string-select.ts) — option lists; single and multi-select.
- [`entity-selects.ts`](./entity-selects.ts) — user, role, channel, mentionable selects.

See also: [buttons](../buttons), [modals](../modals).
